# South Australia (SA) Planning Portal scraper

* Server - .NET
* Cookie tracking - No
* Pagnation - Yes
* Javascript - No
* Ajax - Yes
* Clearly defined data within a record - Yes
* Scrape detail page - No

Setup MORPH_PERIOD for data recovery, available options are

* thisweek (default)
* thismonth
* lastmonth
* year (ie. 2017, min 2003)

Enjoy
